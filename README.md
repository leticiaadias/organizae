# Organizaê - Sistema de Gerenciamento de Eventos 📅

Projeto de TCC focado na organização e gestão de eventos, desenvolvido com uma arquitetura moderna separando Backend e Frontend.

## 🚀 Tecnologias Utilizadas

### Backend

- **Django & Django REST Framework**: API robusta e escalável.
- **PostgreSQL (Supabase)**: Banco de dados relacional na nuvem.
- **Token Authentication**: Segurança no acesso dos usuários.

### Frontend

- **React (Vite)**: Performance e agilidade no desenvolvimento.
- **Tailwind CSS v4**: Estilização moderna e responsiva.
- **React Router Dom**: Navegação entre páginas (Login, Cadastro, Home).

## 🛠️ Como rodar o projeto

### Backend

1. Ative o ambiente virtual: `source venv/bin/activate`
2. Instale as dependências: `pip install -r requirements.txt` (se houver)
3. Inicie o servidor: `python manage.py runserver`

### Frontend

1. Acesse a pasta: `cd frontend`
2. Instale as dependências: `npm install`
3. Inicie o projeto: `npm run dev`
